# DVD Vranjic – Official Website

Ovo je službena web stranica Dobrovoljnog vatrogasnog društva Vranjic.

Projekt je izrađen pomoću modernih web tehnologija i hostan na Vercel platformi.

---

## 🚀 Live Website

🌐 https://dvd-vranjic.hr

---

## 🛠️ Tech Stack

- ⚡ Vite
- ⚛️ React
- 🟦 TypeScript
- 🎨 Tailwind CSS
- 🧩 shadcn/ui
- 🔀 React Router DOM
- ☁️ Vercel (hosting)

---

## 📁 Project Structure

- `src/` – glavni kod aplikacije
- `components/` – UI komponente
- `pages/` ili `app/` – stranice
- `public/` – statički fajlovi
- `vercel.json` – konfiguracija za production routing
